### Status Transitions

**Work Order:** `Draft` вҶ’ `Pending` вҶ’ `Approved` вҶ’ (creates Order) | `Cancelled`

**Order:** `Pending` вҶ’ `Shipping` вҶ’ `Invoiced` вҶ’ `PaymentReceived` вҶ’ `Completed` | `Cancelled`

**Order Item:** `Pending` вҶ’ `Prepress` вҶ’ `Press` вҶ’ `Bindery` вҶ’ `Shipping` вҶ’ `Invoiced` вҶ’ `Completed` | `Hold` | `Outsourced` | `Cancelled`

**Work Order Item:** `Draft` вҶ’ `Pending` вҶ’ `Proofing` вҶ’ `Approved` | `Cancelled`

**Typesetting:** `InProgress` вҶ’ `WaitingApproval` вҶ’ `Completed`
